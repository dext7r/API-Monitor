# API Monitor 2FA 助手 - 浏览器扩展安装教程

此扩展程序可以帮助您在网页登录时自动识别 2FA 输入框，并一键填充验证码。

## 🛠️ 安装步骤

1.  **下载并解压**：
    - 下载 `api-monitor-2fa-extension.zip`。
    - 将 ZIP 文件解压到一个固定位置（例如 `D:\Apps\api-monitor-extension`）。**请勿在安装后删除或移动此文件夹**。

2.  **打开浏览器扩展管理**：
    - 在 Chrome 或 Edge 浏览器地址栏输入 `chrome://extensions` 并回车。
    - 在页面右上角开启 **“开发者模式” (Developer mode)** 开关。

3.  **加载扩展程序**：
    - 点击左上角的 **“加载已解压的扩展程序” (Load unpacked)** 按钮。
    - 在弹出的文件选择框中，选择刚才解压出的文件夹（包含 `manifest.json` 的那个文件夹）。

## ⚙️ 配置说明

1.  点击浏览器工具栏中的扩展图标（如果找不到，点击拼图图标里的“2FA 验证码”）。
2.  点击弹出窗口右上角的 **⚙️ (齿轮图标)** 进入设置页。
3.  在 **服务器设置** 中：
    - **服务器地址**：填写您的 API Monitor 地址（例如 `http://192.168.1.100:5173`）。
    - **管理密码**：填写您在 API Monitor 中设置的 `x-admin-password`。
4.  点击 **保存配置**。

## 💡 使用方法

- **自动填充**：当您在任何网站打开 2FA 验证页面时，输入框右侧会出现 🔐 图标，点击后选择对应账号即可自动填充。
- **快速复制**：点击浏览器工具栏的扩展图标，直接点击账号条目即可自动复制验证码。

---
*提示：如果无法显示验证码，请检查服务器地址和密码是否正确，并确保服务器已开启且可从浏览器所在网络访问。*
